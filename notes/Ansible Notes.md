# Ansible Notes

## Install on Ubuntu from PPA

Update apt cache
`sudo apt update`

Install Pre-reqs
`sudo apt install software-properties-common`

Install Ansible PPA
`sudo apt-add-repository --yes --update ppa:ansible/ansible`

Install Ansible
`sudo apt install ansible`
